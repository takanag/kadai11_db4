# ①課題番号-プロダクト名
8 源泉温泉検索「源泉しか勝たん」

## ②課題内容（どんな作品か）
温泉のなかでも源泉は別格！ でも源泉の温泉がどこにあるのか、検索するのは至難の業。そこでみんながお気に入りの源泉情報を共有する口コミサイト。同じ源泉フリークとして、自分のお気に入りの源泉も共有しよう！
